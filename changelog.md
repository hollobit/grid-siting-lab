## 2026-07-16

### GW STRATEGY · 1 GW+ 최적 입지와 국가 로드맵 배분

- 발전 클러스터 허브를 1.5 GW+ 기준 9곳으로 확장(고리·서인천·한울·보령·당진·광양·한빛·포천·월성)하고, 정부 목표(2029 8.4 GW → 2035 18.4 GW)를 허브별로 배분하는 GW STRATEGY 섹션 추가.
- 배분 모델: 2029 = 클러스터 × 직접연계 10% ÷ 1.25(N+1), 2035 = 계통 보강·신규 전원 병행 20% 프록시. 합계와 목표 대비 **부족분을 명시**(2029 5,061/8,400 MW · 부족 3,339 / 2035 10,121/18,400 MW · 부족 8,279)하고 충당 수단(SMR·해상풍력·LNG 직결 신규 전원, HVDC 동해안—수도권, 호남 RE100 캠퍼스, 한전 실데이터 연동 후 재배분)을 제시.
- **1 GW+ 최적 추천**: 2035 프록시 기준 1 GW 이상 가능 허브 중 적합도 최고 — 고리원자력 인접(적합도 82, 클러스터 10,593 MW, IT ~1,695 MW). 해결 과제(765 kV/HVDC 인입·계통영향평가 패스트트랙·백본 전용선·산단 지정·해수냉각·신규 전원·단계 증설) 병기.
- 1·2단계 달성 경로 서술(서해안 화력벨트+동남권 원전벨트 병행 착공 → 동해안 HVDC·SMR·해상풍력 확장, 호남 RE100 존, 해수·재이용수 냉각 표준화). 가중치 변경 시 1 GW+ 추천 재계산.

### 초대형 신설 후보 · GW급 발전원 직접연계형 스크리닝

- 계통 여유 프록시로는 초대형(300 MW) 전제 충족 입지가 없어, **GW급 발전 클러스터 인접 신설** 경로를 별도 트랙으로 추가: OSM 태그 800 MW+ 발전소를 앵커로 반경 15 km 클러스터 용량을 합산, 2.5 GW 이상·40 km 분산 조건으로 상위 5곳 선정.
- 공급 모델: 전용선로·직접 PPA로 클러스터 용량의 10%(프록시)를 공급 → N+1 기준 최대 IT 환산. 결과(2026-07-16 기준): 고리원자력 인접 10,593 MW → IT ~847 MW, 서인천복합 9,119 → ~730, **한울원자력(울진) 7,884 → ~630**, 보령화력 7,373 → ~590, 당진화력 6,655 → ~533 MW — 5곳 모두 300 MW 충족.
- "다른 조건 개선" 정량화: 취약 요인(백본·용수·토지·주거·물류·인재·재해)별 개선 과제 칩과, 개선 목표치(72) 달성 시 잠재 점수(현재 → 개선 시) 표시. 예: 한울 인접 68 → 74 (백본 전용선·해수 냉각·산단 지정·주거 인프라·원격 운영 분리).
- 지도 SUITABILITY 레이어에 주황 대형 점선 링으로 표시, 클릭 시 즉석 분석. 가중치 변경 시 점수 재계산. 직접연계 비율·개선 목표치는 협상·투자 전제의 탐색용 프록시로 고지.

## 2026-07-15

### 규모별 전국 추천 TOP 5 지도 표시

- **시드 생성**: 전국 345 kV+ 송전선 way의 중간점을 0.18°(~20 km) 셀로 중복 제거해 133개 시드 지점 생성 — 계통 인접·육상 지점만 후보가 되도록 설계.
- **채점**: 각 시드를 임의 지점 분석 엔진(OSM 반경 50 km 실측 + 거리 프록시 + 재해존 회복력)으로 채점. 초기 렌더 후 12개 단위 배치로 백그라운드 실행(~1초), 완료 토스트 표시.
- **클래스별 TOP 5**: 큐레이션 후보 + 시드 풀에서 전제 충족(전력·용수 공급 + 민원 페널티 < 12) 우선 정렬, 30 km 최소 간격 그리디 선정으로 지리적 분산 보장. 전제 미충족 후보는 △(보강 전제)로 구분.
- **지도 표시**: SUITABILITY 레이어에 클래스별 색상·크기의 점선 링 마커(초대형 보라 > 대형 핑크 > 중형 시안 > 소형 초록, 1위는 굵은 테두리). tooltip에 순위·점수·공급 한도·민원 페널티, 클릭 시 해당 지점 즉석 분석. 범례에 4개 클래스 링 추가.
- **연동**: SCALE MODEL FIT 카드에 "전국 추천 TOP 5" 목록 표시, 가중치(POLITICAL/BALANCE 포함) 변경 시 재순위(시드 채점은 재사용).

### 정치·균형발전 가중치 + 규모별 AIDC 모델 + 리포트/재해/한전 연동 (미배포 → 이번 커밋)

- **SCORING ASSUMPTIONS 확장**: POLITICAL(정치적 고려 · 지자체 유치 의지/정책 정합성/규제 리스크 프록시)과 BALANCE(지역균형발전 · 비수도권 가산 프록시) 가중치 슬라이더 추가 (기본 10%/10%, 범위 0–25%). 후보지별 political/balance 요인 점수 수록(이천은 수도권 규제로 34/25), 요인 리스트 12개·모델 구성 바 10개로 확장. 임의 지점은 political 55 중립 + 수도권 거리 기반 balance.
- **SCALE MODEL FIT (규모별 AIDC 적합도)**: 초대형(300 MW)/대형(100 MW)/중형(40 MW)/소형(10 MW) 4개 모델별 점수 = 기본 적합도 × (0.45 + 0.55 × 공급충족률) − 민원 페널티. 공급 한도는 표준구성(N+1·하이브리드·부하율 80%·PUE 1.25) 기준 계통 수전 vs 냉각 용수 병목값. 민원 페널티는 토지·인허가 취약, 용수 스트레스, 주민 전력 경합(발전량 점유율), 대도시 인접을 규모 계수(초대형 ×1.4)로 증폭. **전력·용수 충족 + 민원 페널티 < 12 전제를 만족하는 후보 중 최고점**을 모델별 추천 입지로 표시.
- **후보지 비교 리포트 PDF**: SHORTLIST 섹션 "리포트 PDF" 버튼 → 후보지(선택한 임의 지점 포함) × 규모 모델 매트릭스, 실측(발전량·최근접 345 kV+), 공급 한도·병목, 용수·탄소, 한전 연동 상태, 모델별 추천 입지, 가중치·시나리오 스냅샷을 인쇄 뷰로 생성해 브라우저 "PDF로 저장"으로 내보내기.
- **재해(홍수·지진) 레이어**: HAZARD 지도 토글 추가 — 홍수 저지대 프록시존 5곳(한강·금강·영산강·낙동강 하류, 삽교천 유역)과 지진 다발권 3곳(경주 2016 M5.8 · 포항 2017 M5.4 · 양산단층대). 임의 지점의 재해·회복력 점수를 중립값(60) 대신 재해존 근접도 기반 산정으로 교체. 공공데이터(홍수위험지도·기상청 지진) 연동 전 탐색용 프록시로 고지.
- **한전 접속여유 연동 구조**: `data/kepco_capacity.json` 존재 시 자동 로드 — 후보지 반경 30 km 최근접 변전소 여유용량으로 수전 여유(headroom)를 교체하고 시나리오 카드에 변전소명·여유 MW·기준일 표시. 스키마와 사용법은 `data/kepco_capacity.sample.json` 참조. 미연동 시 "미연동 · 스크리닝 프록시" 라벨.

### 초기 구축 + OSM 전력망 스냅샷 + 배포 (a4bc59d)
